# Shift Earnings Tracker

A Python program that allows you to track and record what you make during each shift and view your total earnings over time.

## Features
- Record earnings from each shift
- View total earnings
- Track income over multiple days

## Built With
- Python

## Purpose
This project was created to practice Python while building a practical tool for tracking shift-based income.
## How to Run

1. Make sure Python is installed
2. Download the file
3. Run the program:

```bash
python budget_tracker.py
